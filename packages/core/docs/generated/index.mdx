**core** is a TypeScript documentation toolkit that performs a single AST traversal of your project and produces API docs, OpenAPI specs, executable doctests, and AI context files in one pass.

## Features

- Full TypeScript support with TSDoc extraction
- Auto-generated API reference from source code
- Executable `@example` blocks as doctests
- AI-ready context files from a single build pass

## Installation

```bash
npm install -D @codluv/llmtxt
```

## Quick Example

```ts
encodeBase62(0);   // "0"
encodeBase62(61);  // "z"
encodeBase62(62);  // "10"
```
