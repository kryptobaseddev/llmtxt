## LRUCache

Generic least-recently-used (LRU) cache with time-to-live support.

**Signature**

```typescript
typeof LRUCache
```

**Example**

```ts
const cache = new LRUCache<string>({ maxSize: 100, ttl: 60_000 });
cache.set('key', 'value');
cache.get('key'); // "value"
```

**Methods**

### get()

Retrieve a cached value by key.

```typescript
(key: string) => T | undefined
```

### set()

Insert or update a cache entry.

```typescript
(key: string, value: T, ttl?: number) => void
```

### delete()

Remove a single entry from the cache by key.

```typescript
(key: string) => void
```

### clear()

Remove all entries from the cache and reset hit/miss statistics.

```typescript
() => void
```

### size()

Return the number of live (non-expired) entries in the cache.

```typescript
() => number
```

### has()

Check whether a non-expired entry exists for the given key.

```typescript
(key: string) => boolean
```

### getStats()

Retrieve a snapshot of cache performance statistics.

```typescript
() => CacheStats
```

### resetStats()

Reset the hit/miss counters to zero without clearing cached entries.

```typescript
() => void
```

## computeSignature(params, secret)

Compute the HMAC-SHA256 signature for a set of signed URL parameters.

**Signature**

```typescript
(params: SignedUrlParams, secret: string) => string
```

**Parameters**

| Name     | Type              | Description                                                    |
| -------- | ----------------- | -------------------------------------------------------------- |
| `params` | `SignedUrlParams` | The signed URL parameters to include in the signature payload. |
| `secret` | `string`          | The shared HMAC secret.                                        |

**Returns** — A 16-character hex string representing the truncated HMAC signature.

**Example**

```ts
const sig = computeSignature(
  { slug: 'xK9mP2nQ', agentId: 'agent-1', conversationId: 'conv_1', expiresAt: 1711234567890 },
  'my-secret',
);
// sig.length === 16
```

## generateSignedUrl(params, config)

Generate a signed URL for accessing a document.

**Signature**

```typescript
(params: SignedUrlParams, config: SignedUrlConfig) => string
```

**Parameters**

| Name     | Type              | Description                                                    |
| -------- | ----------------- | -------------------------------------------------------------- |
| `params` | `SignedUrlParams` | The signed URL parameters (slug, agent, conversation, expiry). |
| `config` | `SignedUrlConfig` | The HMAC secret and base URL for URL construction.             |

**Returns** — The fully-qualified signed URL string.

**Example**

```ts
const url = generateSignedUrl(
  { slug: 'xK9mP2nQ', agentId: 'my-agent', conversationId: 'conv_123', expiresAt: Date.now() + 3600000 },
  { secret: 'shared-secret', baseUrl: 'https://llmtxt.my' },
);
// => "https://llmtxt.my/xK9mP2nQ?agent=my-agent&conv=conv_123&exp=1711234567890&sig=a1b2c3d4e5f6a7b8"
```

## generateTimedUrl(params, config, ttlMs)

Generate a signed URL that expires after the given duration.

**Signature**

```typescript
(params: Omit<SignedUrlParams, "expiresAt">, config: SignedUrlConfig, ttlMs?: number) => string
```

**Parameters**

| Name     | Type                   | Description                                                                |
| -------- | ---------------------- | -------------------------------------------------------------------------- |
| `params` | `Omit<SignedUrlParams` | Slug, agentId, and conversationId (expiresAt is calculated automatically). |
| `config` | `SignedUrlConfig`      | The HMAC secret and base URL for URL construction.                         |
| `ttlMs`  | `: number`             | Time to live in milliseconds (default: 1 hour / 3 600 000 ms).             |

**Returns** — The fully-qualified signed URL string with a computed expiration.

**Example**

```ts
const url = generateTimedUrl(
  { slug: 'xK9mP2nQ', agentId: 'agent-1', conversationId: 'conv_1' },
  { secret: 'secret', baseUrl: 'https://llmtxt.my' },
  300_000, // 5 minutes
);
```
