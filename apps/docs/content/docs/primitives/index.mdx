# Shared Primitives

The `llmtxt` package exports seven stable subpaths under the **Shared Primitives Subpath Contract** (v2026.4.9, T550). Each subpath is:

- **Tree-shakeable** — importing one subpath does not load the others.
- **Independently versioned** — breaking changes to any subpath surface trigger a minor bump per `STABILITY.md`.
- **Contract-tested** — `packages/llmtxt/src/__tests__/subpath-contract.test.ts` validates every export on every CI run.
- **CI-guarded** — `scripts/check-subpath-exports.sh` fails a pull request if any contracted export is removed without a corresponding version bump.

## Quick Reference

| Subpath | Key exports | Use when… |
|---|---|---|
| `llmtxt/sdk` | `AgentSession`, `DocumentLifecycle`, `VersionStack`, `ConsensusEvaluator` | You need the full collaborative document workflow |
| `llmtxt/identity` | `AgentIdentity`, `signRequest`, `verifyRequestSignature` | You need to sign or verify agent requests |
| `llmtxt/blob` | `hashBlob`, `blobNameValidate`, `FilesystemBlobStore` | You need to store or transfer binary files |
| `llmtxt/events` | `EventBus`, `createEventBus` | You need a typed event bus for document streams |
| `llmtxt/crdt` | `crdtNewDoc`, `crdtApplyUpdate`, `crdtMergeUpdates` | You need real-time section collaboration |
| `llmtxt/transport` | `UnixSocketTransport`, `HttpTransport` | You need direct P2P agent-to-agent communication |
| `llmtxt/similarity` | `textSimilarity`, `rankBySimilarity`, `minhashFingerprint` | You need to rank or deduplicate text content |

## Install

```bash
npm install llmtxt
```

All 7 subpaths are included in the base package — no additional installs required for the primitives themselves.

```ts
// Each import is independent
import { AgentSession } from 'llmtxt/sdk';
import { AgentIdentity } from 'llmtxt/identity';
import { hashBlob } from 'llmtxt/blob';
import { createEventBus } from 'llmtxt/events';
import { crdtNewDoc } from 'llmtxt/crdt';
import { UnixSocketTransport } from 'llmtxt/transport';
import { textSimilarity } from 'llmtxt/similarity';
```

## Stability Policy

The subpath contract is defined in `packages/llmtxt/STABILITY.md`. Key rules:

- Any **removal or rename** of a contracted export is a **breaking change** — requires a major version bump.
- Any **signature change** (parameter added without a default, return type narrowed) is a **breaking change**.
- **Adding** new exports to an existing subpath is backward-compatible and requires only a patch bump.
- Subpaths marked `@experimental` in JSDoc are excluded from the contract and may change freely.
